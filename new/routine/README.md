# 锦程数码会员小程序（Jingcheng Digital Membership Mini-Program）

本项目为**锦程数码**官方会员小程序的微信原生开发版本（代码目录：`new/routine`）。项目已全面对齐 **Desktop 最终版设计图（24日UI）**，完成全量核心业务页面的视觉与交互重构，呈现出统一、高端的**轻奢会员风**。

---

## 一、项目说明（Project Description）

### 1.1 业务定位
锦程数码会员小程序是连接数码3C消费者与线下门店的核心纽带。通过该小程序，会员可以：
- 申请并绑定固定权益档位。
- 积累消费积分，并在积分商城兑换专享好礼。
- 获得并核销现金券，享受门店专属折扣。
- 绑定专属客户经理，享受一对一专业数码服务。

同时，小程序内置了**店员工作台**、**店长审批中心**与**商家工作台**，实现了门店店员、店长及合作商家的移动端闭环管理与快速核销。

### 1.2 视觉与交互规范
项目严格遵循 **Desktop 最终版设计图** 的像素级视觉规范，彻底摒弃了传统的促销红/满减黄，确立了极具身份感与克制美学的视觉基调：
- **主色基调**：深色系渐变基调（深蓝 `#1A1F36` 到 `#2D3561`）作为头部与核心卡片背景，彰显高端会员身份感。
- **点缀色彩**：品牌金（`#C9A227` / `#D4AF37`）用于权益、积分、价格、核心数值与主操作按钮。
- **背景底色**：统一采用暖灰底（`#F5F3F0`），营造温暖、干净、克制的浏览氛围。
- **卡片设计**：大面积白卡（`#FFFFFF`） + 柔和圆角（`24rpx`） + 极浅投影（`0 4rpx 24rpx rgba(0,0,0,0.04)`）。
- **交互反馈**：按钮及可点击卡片均配置微缩回弹反馈（`hover-class`），提供极佳的原生手感。

## 四、编译与部署指南（Build & Deploy）

由于本项目为原生微信小程序，您可以通过以下步骤在本地进行预览与上传：

1. **导入项目**：打开 **微信开发者工具**，导入目录 `new/routine`。
2. **本地编译**：点击开发者工具中的 **「编译」** 按钮，即可在模拟器中预览重构后的轻奢会员风页面。
3. **上传发布**：点击右上角 **「上传」** 按钮，输入版本号和备注，上传至微信公众平台，并在后台将其设为体验版进行验收。
