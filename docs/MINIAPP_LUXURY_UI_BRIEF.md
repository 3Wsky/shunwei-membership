# 顺为小程序 · 轻奢会员风 UI 重构设计简报 v1.0

> 派单：产品经理 → UX/UI 设计师 | 工程：`shunwei-app-v2` | 2026-06-20

## 一、整体视觉定位

**「轻奢会员风，不是廉价促销感。」**

| 维度 | 规范 |
|------|------|
| 主色 | 深色系基调（深蓝 `#1A1F36` / 深灰 `#2D2D2D`） |
| 点缀 | 品牌金 `#C9A227` / `#D4AF37`（权益、积分、价格） |
| 氛围 | 参考星巴克 / 京东 PLUS — 身份感、干净克制 |
| 禁忌 | 大红大紫电商促销风、满屏标签、闪烁动效 |
| 卡片 | 大面积白卡 + 柔和圆角 `24rpx` + 极浅阴影 `0 4rpx 24rpx rgba(0,0,0,0.06)` |
| 数字 | 积分/等级/金额 — 大号粗体 + 金色 |

### Design Token 建议

```scss
$color-bg-page: #F5F3F0;
$color-bg-dark: linear-gradient(135deg, #1A1F36 0%, #2D3561 100%);
$color-gold: #C9A227;
$color-gold-light: #D4AF37;
$color-text-primary: #1A1A1A;
$color-text-muted: #8C8C8C;
$radius-card: 24rpx;
$shadow-card: 0 4rpx 24rpx rgba(0,0,0,0.06);
```

## 二、首页（商品兑换展示）

**「精品店浏览，不是批发市场。」**

- 顶部 **磁吸会员身份条**：深色渐变背景，头像 + 等级徽章 + 当前积分，滚动吸附
- 商品区 **两列瀑布流**：大图露出、名称单行、积分价金色标签、「会员专属」角标
- 卡片点击：轻微上浮 + 缩放（参考 App Store）
- 无图占位：柔和渐变 + 产品名，保持干净

**涉及文件**：`pages/index/index.vue`、`pages/products/list.vue`

---

## 三、积分兑换 + 核销流程

**「兑换有仪式感，核销有信任感。」**

| 环节 | 要求 |
|------|------|
| 商品详情 | 图占上半屏；所需积分大号金色；按钮文案「立即兑换」非「购买」 |
| 确认弹窗 | 底部半屏滑出；展示消耗 vs 剩余积分对比 |
| 核销码卡 | 大二维码 + 淡金光晕背景；待核销/已核销徽标 |
| 核销记录 | 电影票券样式；已核销灰色蒙层，未核销鲜活色彩 |

**涉及文件**：`pages/integral/mall.vue`、`pages/integral/index.vue`、`components/SwQrCode/`

---

## 四、个人中心

**「数据看板感，不是设置表单。」**

- 顶部大卡片：头像 + 昵称 + **等级徽章图形**（非纯文字）
- 数据面板一排：积分余额 / 优惠券数 / 待核销笔数 — 数字大而清晰
- 菜单：圆角卡片分组（我的订单、核销记录、优惠券、等级权益），右箭头整行可点
- 等级页：阶梯式权益展示；当前等级高亮放大；未解锁灰度锁图标

**涉及文件**：`pages/user/index.vue`、`pages/member/center.vue`、`pages/member/purchase.vue`

---

## 五、交互动效（关键）

| 场景 | 动效 |
|------|------|
| 页面切换 | 平滑推入，非生硬跳转 |
| 按钮点击 | 微缩回弹（微信原生手感） |
| 积分变动 | 数字滚动递增/递减 |
| 核销码出现 | 缩放弹出 + 轻光晕扩散 |
| 列表滚动 | 卡片逐个淡入上移 |
| 空状态 | 插画 + 「去兑换第一件商品吧」引导按钮 |

> 注：微信 AI 原子组件不支持动画；动效仅用于普通小程序页面。
