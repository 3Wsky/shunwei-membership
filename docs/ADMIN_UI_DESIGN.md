## 三、布局规范 v3.0（CRMEB 参考 · 已实现）

> **色彩决策**：顶栏采用品牌深靛 `#1A1A2E`（非 CRMEB `#1890ff`），激活 Tab 用玫瑰珊瑚 `#E8436E` 底+下划线，保留 FZLSaas 品牌识别同时对齐 CRMEB **顶栏 Tab + 白侧栏 + 卡片内容区** 结构。

### 3.1 整体结构

```
┌──────────────────────────────────────────────────────────────────────────┐
│ 顶栏 72px · 渐变色（#002c8c -> #40a9ff）                                  │
│ 锦程会员电商系统  │ [数据][会员][运营][商家][系统]（圆角） │ 刷新 用户 ▼  │
├──────────┬───────────────────────────────────────────────────────────────┤
│ 白侧栏   │ 内容区 #F0F2F5 padding 16px                                   │
│ 200px    │ ┌─ PageShell 白卡片 ─────────────────────────────────────┐  │

│          │ └─────────────────────────────────────────────────────────┘  │
└──────────┴───────────────────────────────────────────────────────────────┘
```

### 3.2 一级模块 Tab → 二级侧栏映射

| 一级 Tab | 侧栏路由 | 页面 |
|----------|----------|------|
| **数据** | 数据看板 | `/dashboard` |
| **会员** | 会员管理、店员管理 | `/members`, `/staff` |
| **运营** | 审批管理、积分商城、新客抽奖 | `/approval`, `/integral-mall`, `/lottery` |
| **商家** | 商家管理、商品管理 | `/merchant`, `/products` |
| **系统** | 审计日志 | `/audit-logs` |

### 3.3 PageShell 模板（列表页标准）

| 槽位 | 用途 | CRMEB 对应 |
|------|------|-----------|

| key | 标题 | 左边框色 | 图标 | 数据来源 |

### 9.4 快捷入口条

白底卡片，横向 `el-space`，每项为 `el-link` + 数字徽章：

| 入口 | 跳转 | 条件显示 |
|------|------|----------|
| 待审批 N 条 | `/approval?status=pending_admin` | N>0 时主色高亮 |
| 待结算 ¥X | `/merchant?tab=settlement` | R2，R1 可灰显 |
| 今日通过 N 条 | `/approval?status=approved&date=today` | 始终显示 |
